## Getting started
- We recommend `yarn` due to `resolutions`
- Add a `.env` to root
- Populate these values
```
CLUSTER=mainnet-beta
WALLET_PRIVATE_KEY=<wallet private key> ## or set it up in src/constants
```
- yarn
- yarn start
